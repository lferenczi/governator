Governator-mini is a simplified version of Netflix's Governator library, providing an extensions and utilities that extend Google Guice:

  • Lifecycle management
  • Configuration to field mapping
  • Parallelized object warmup
  • Lazy singleton support
  • Generic binding annotations

See https://github.com/lferenczi/governator-mini for details. See also the original: https://github.com/Netflix/governator for details.
